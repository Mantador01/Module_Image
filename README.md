# Module Image

Projet réalisé pour le devoir de l'UE LIFAPCD 2022/2023

Pour accéder au programme du module image il suffit de réaliser ses quelques étapes :

1) Tout d'abord ce positionner dans le fichier module image avec le terminale.

2) Taper cette commande dans le terminale -> make clean

3) Ensuite taper celle ci -> make

4) Après que toute les exécutions ont été faites vous pouvez lancer le programme en tapant -> ./bin/affichage

Voilà, le programme devrais ouvrir une fenêtre faisant apparaitre un petit personnage vert avec une canne ! 

Pour l'interactivité vous avez :
ECHAPPE : pour fermer la fenêtre.
T : Pour zoomer.
G : Pour dézoomer.
